## Basic usage

Pass in a value and get a boolean telling you if the value is an Object.

```ts
import { isObject } from 'radash'

isObject('hello') // => false
isObject(['hello']) // => false
isObject(null) // => false
isObject({ say: 'hello' }) // => true
```
